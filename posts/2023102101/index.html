Debian11上添加交换空间

debian11如何添加swap空间

Oct 21, 2023
阅读时长: 1 分钟

新增Swap空间

  1. 检查当前的交换空间： 以下命令检查当前系统中是否已经存在交换分区或交换文件。

    swapon --show

    如果没有显示任何交换空间，请继续以下步骤。

  2. 创建交换文件 (Swap File)： 选择一个适当的位置和文件名（例如 /swapfile），并选择要创建的大小。通常，交换文件的大小可以设置为物理内存的1.5倍或2倍。

    sudo fallocate -l 4G /swapfile

    如果系统不支持 fallocate 命令，可以使用 dd 命令：

    sudo dd if=/dev/zero of=/swapfile bs=1M count=2048

    这将创建一个大小为2GB的交换文件。

  3. 设置文件权限： 为了确保文件的安全性，你需要限制交换文件的访问权限。

    sudo chmod 600 /swapfile

  4. 格式化为交换空间： 使用以下命令将文件转换为交换空间。

    sudo mkswap /swapfile

  5. 启用交换空间： 启用新创建的交换文件。

    sudo swapon /swapfile

  6. 永久性配置： 如果你希望在系统重新启动后保持交换空间的设置，你需要将它添加到 /etc/fstab 文件中。

    echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

  7. 验证交换空间：

    swapon --show

交换比例

  1. 查询当前比例：

该值通常在0到100之间。默认情况下，Debian系统的交换比例设置为60。

cat /proc/sys/vm/swappiness

或

sysctl vm.swappiness
  1. 修改当前比例：
echo "vm.swappiness=0" | sudo tee -a /etc/sysctl.conf

移除交换空间

  1. 禁用交换空间：

    sudo swapoff /swapfile

  2. 删除交换文件或分区：

    sudo rm /swapfile

  3. 从 /etc/fstab 中移除条目：

    sudo nano /etc/fstab

  4. 重新加载系统配置：

    sudo sysctl --system
