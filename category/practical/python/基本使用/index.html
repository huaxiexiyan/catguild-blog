虚拟环境#

1、安装

pip install pipenv

2、（选用）修改环境变量 该环境会让虚拟环境创建再当前项目下，否则会创建默认位置

PIPENV_VENV_IN_PROJECT=1

3、到项目目录下

pipenv install

该命令会创建虚拟环境目录，并生成 Pipfile 和Pipfile.lock 文件

4、安装依赖

pipenv install package_name
